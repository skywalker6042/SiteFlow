# SiteFlow iOS App

This folder contains the starting Swift code for the SiteFlow iOS app using SwiftUI.

## Setup in Xcode

1. On a macOS machine with Xcode installed, create a new project:
   - Open Xcode
   - File > New > Project
   - Choose "App" under iOS
   - Interface: SwiftUI
   - Language: Swift
   - Name: SiteFlowApp
   - Create

2. Replace the generated App.swift and ContentView.swift with the files from this folder.

3. Build and run the app in the simulator.
